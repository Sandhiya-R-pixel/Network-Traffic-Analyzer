# Network-Traffic-Analyzer
Project Title:
Network Traffic Analyzer using Wireshark & Scapy for Smart City Management

Description:
This project analyzes real-time network traffic using Python’s Scapy and Wireshark tools.
It helps monitor data flow, detect unusual activity, and improve the reliability of smart city communication systems — supporting SDG 11: Sustainable Cities and Communities.

Key Features:

Captures and analyzes live network packets

Displays IP addresses, protocols, and traffic details

Detects abnormal or suspicious activity

Lightweight, easy to use, and educational

Tools Used:
Python 3.12, Scapy, Wireshark, Thonny IDE

Purpose:
To enhance smart city network management by ensuring secure, efficient, and sustainable data communication.
